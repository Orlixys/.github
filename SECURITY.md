# Política de Segurança - Orlixys

A Orlixys leva a segurança a sério. Agradecemos qualquer esforço para descobrir e reportar vulnerabilidades de forma responsável.

## Versões Suportadas

Atualmente, fornecemos atualizações de segurança para as seguintes versões:

| Versão | Suportada |
| ------- | :-------: |
| 1.0.x   | ✅        |
| < 1.0   | ❌        |

## Reportando uma Vulnerabilidade

**Por favor, não abra uma Issue pública para reportar uma vulnerabilidade.**

Se você encontrar um problema de segurança, siga os passos abaixo:
1. Envie um e-mail detalhado para: `security@orlixys.com`.
2. Inclua o máximo de informações possível (passos para reproduzir, impacto, capturas de tela).
3. Aguarde uma resposta em até 48 horas antes de qualquer divulgação pública.

## Nossa Promessa
- Reconheceremos o recebimento do seu relatório rapidamente.
- Trabalharemos para corrigir a falha o mais rápido possível.
- Daremos o devido crédito pelo achado assim que a correção for publicada.
